# ボムへいをわけろ！
## 実行環境の必要条件
* python >= 3.10
* pygame >= 2.1

## ゲームの概要
一定間隔ごとに出現するボムへい！
正しく選別して爆発しないように仕分けよう！

## ゲームの実装
###共通基本機能
* 背景画像の読み込み
### 担当追加機能
* ボム兵が反射する機能を付け足す(担当：松元)
*音楽、メニュー画面
*ボムの生成、ランダムな動き
*安全地帯の生成
